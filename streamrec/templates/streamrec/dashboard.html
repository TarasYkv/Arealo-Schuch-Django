StreamRec Bedienungsanleitung
Erste Schritte
  1. Studio öffnen: Klicken Sie auf "Aufnahme starten" um das Studio zu öffnen
  2. Browser-Berechtigungen: Erlauben Sie Zugriff auf Kamera und Mikrofon
  3. Stream starten: Nutzen Sie "Kamera starten" oder "Bildschirm teilen"
  4. Aufnahme beginnen: Rote Aufnahme-Taste für 3-Minuten-Recording
Interface Überblick
  • Links: Stream-Kontrollen & System-Info
  • Mitte: Live-Vorschau Canvas (9:16)
  • Unten: Aktive Streams & Layout-Optionen
  • Mobile: Automatisches responsive Layout
Kamera-Stream
  • Starten: "Kamera starten" Button in der linken Sidebar
  • Status: Grüner Punkt = aktiv, Grauer Punkt = inaktiv
  • Qualität: HD 1280x720 mit Audio-Aufnahme
  • Toggle: Klick auf Thumbnail zum Ein-/Ausblenden
Bildschirm-Streams
  • Erster Bildschirm: "Bildschirm teilen" Button
  • Weitere Bildschirme: "Weiteren Bildschirm hinzufügen"
  • Unbegrenzt: So viele Monitore wie gewünscht
  • Entfernen: ✕ Button auf Stream-Thumbnail
Stream-Einstellungen
  • Display-Modi:
    Vollbild, Gestreckt, Angepasst, Beschnitten
  • Aktivierung:
    Grüner Rand = aktiv, Gelber Rand = inaktiv
  • Performance:
    FPS-Counter & Drop-Rate Monitoring
Tipp: Auf Mobilgeräten ist Bildschirm-Sharing nicht verfügbar. Der Button wird automatisch deaktiviert.

Aufnahme-Prozess
  1. Streams aktivieren: Starten Sie gewünschte Kamera/Bildschirm-Streams
  2. Layout wählen: PiP, Nebeneinander oder Freies Layout
  3. Aufnahme starten: Roter "Aufnahme starten" Button
  4. 3-Sekunden Countdown: Bereiten Sie sich vor
  5. Recording läuft: Max. 3 Minuten HD-Aufnahme
  6. Stoppen: Automatisch oder manuell beenden
Export-Optionen
Nach der Aufnahme:
  • Vorschau ansehen
    Video vor dem Export prüfen
  • Herunterladen
    WebM-Datei lokal speichern
  • Auf Server speichern
    Mit benutzerdefinierten Titel
Wichtige Hinweise:
  • Maximale Aufnahmedauer: 3 Minuten
  • Format: WebM (H.264/VP9 je nach Browser)
  • Auflösung: 9:16 Portrait-Format für Social Media
  • Während der Aufnahme werden UI-Elemente ausgeblendet
