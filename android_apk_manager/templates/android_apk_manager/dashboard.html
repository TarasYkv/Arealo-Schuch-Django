Schnellstart
Wie verwende ich den APK Manager?
  1. App erstellen: Klicke auf "Neue App" und gib Package Name und Name ein
  2. Version hochladen: Gehe zum Admin-Bereich und lade eine APK-Datei hoch
  3. Als öffentlich markieren: Setze "Is Public" auf "Ja" damit die App heruntergeladen werden kann
  4. Update-System: Nutze die REST API für automatische Updates in deiner Flutter-App
API-Endpunkt für Flutter Apps:
