Integration Code

Verwenden Sie diese Template-Tags, um LoomAds in Ihre Templates zu integrieren:

1. Template Tags laden
{% templatetag openblock %} load loomads_tags {% templatetag closeblock %}
2. CSS einbinden
{% templatetag openblock %} loomads_css {% templatetag closeblock %}
3. Werbezone anzeigen
{% templatetag openblock %} show_ad_zone 'zone_code' css_class='custom-class' {% templatetag closeblock %}
4. Modal/Pop-up anzeigen
{% templatetag openblock %} show_ad_modal 'zone_code' delay=5000 {% templatetag closeblock %}
5. Statistiken abrufen
{% templatetag openblock %} ad_stats 30 as stats {% templatetag closeblock %}
Impressions: {% templatetag openvariable %} stats.impressions {% templatetag closevariable %}
Klicks: {% templatetag openvariable %} stats.clicks {% templatetag closevariable %}
CTR: {% templatetag openvariable %} stats.ctr {% templatetag closevariable %}%
