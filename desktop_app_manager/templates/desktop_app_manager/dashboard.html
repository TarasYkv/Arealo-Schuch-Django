Schnellstart
Wie verwende ich den Desktop App Manager?
  1. App erstellen: Klicke auf "Neue App" und gib Identifier und Name ein
  2. Version hochladen: Lade eine EXE-Datei hoch und vergib eine Versionsnummer
  3. Als öffentlich markieren: Setze "Öffentlich" auf "Ja" damit die App heruntergeladen werden kann
  4. Update-System: Nutze die REST API fürautomatische Update-Checks
